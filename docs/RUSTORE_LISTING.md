## Краткое описание

Пульт оператора трансляций: вход по токену/пин-коду, счёт, таймер, события матча и управление стримом.

## Полное описание

WAAF Streamer — мобильный операторский пульт для прямых трансляций футбольных матчей.

Возможности:
- вход в турнир по ссылке/токену;
- вход в конкретный матч по PIN-коду;
- управление счётом, таймером и событиями матча;
- запуск и остановка RTMP-трансляции;
- работа с составами и протоколом.
