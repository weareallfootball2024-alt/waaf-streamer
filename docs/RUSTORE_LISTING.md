## Краткое описание

Два режима: пульт трансляций (токен/PIN) и тестирование навыков (вход тренера/админа клуба, запись видео).

## Полное описание

WAAF Streamer — приложение для клубов и операторов турниров.

**Стриминг** — операторский пульт матча:
- вход в турнир по ссылке/токену;
- вход в матч по PIN-коду;
- счёт, таймер, события, RTMP-эфир, составы.

**Тестирование** — для тренера и админа клуба:
- вход по аккаунту WAAF;
- выбор теста и игрока, инструкция по фиксации камеры;
- запись и отправка видео на сервер.
